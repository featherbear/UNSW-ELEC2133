    Circuit Configurations

    • Common source
      • (Most common configuration)
      • Source is common to the input and output
      • 180 degree phase change
      • Medium input impedance
      • Medium output impedance
    • Common gate
      • Gate is common to the input and output
      • No phase change
      • Low input impedance
      • High output impedance

     

    • Common drain
      • "Source follower"
      • Drain is common to the input and output
      • No phase change
      • High input impedance
      • Low output impedance

    • Disable voltage and current sources
    • Mid, High -> Short external capacitor
    • Replace BJT and MOSFET with small signal
    • Low, Mid -> Open internal capacitor
    • High -> Keep internal capacitor
